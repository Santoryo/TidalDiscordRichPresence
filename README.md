# TidalRichPresence
 TIDAL Rich Presence for Discord without Last.FM

<img src="https://i.imgur.com/5hRCJEk.png" width=30%>

## Setup
1. Download from Releases
2. Launch `Tidal Rich Presence.exe`. You will have to login into your TIDAL account. The token is valid for 7 days and it's stored in `token.brel`
